[size=4][b]Enhanced Console Autocomplete[/b][/size]

Adds intelligent autocomplete to the developer console. Big thanks to lStewieAl for the help and ideas in producing this.

[size=3][b]Features[/b][/size]
[list]
[*]Command name autocomplete with parameter hints
[*]Cell names (coc/centeroncell)
[*]Game settings with live value preview (setgs/getgs)
[*]Actor values with live value preview (setav/getav/modav/forceav)
[*]Quests, quest stages, and quest objectives
[*]Quest variables (questname.varName)
[*]Perks (addperk/removeperk/hasperk)
[*]Notes (addnote/addnotens)
[*]Factions (addfaction/removefaction/setfactionrank/setally/setenemy)
[*]Sounds (playsound/playsound3d)
[*]Image space modifiers (imod/rimod)
[*]Weathers (fw/forceweather/sw/setweather)
[*]World spaces (cow/centeronworld)
[*]Idle animations (playidle/forceplayidle)
[*]Music types (playmusic/ismusicplaying)
[*]Form lists (addformtoformlist/removeformfromformlist/dumpformlist/isinlist)
[*]Spells (addspell/removespell/cast/dispel/isspelltarget)
[*]Form type codes for the search command
[*]Base form autocomplete with smart filtering:
[list]
[*]Inventory items for additem/removeitem/drop (weapons, armor, ammo, aid, misc, etc.)
[*]Equippables for equipitem/unequipitem (weapons and armor only)
[*]Placeables for placeatme/placeleveled (NPCs, creatures, objects, furniture, etc.)
[/list]
[*]Command aliases - define shortcuts in the INI file
[*]Ctrl+R reverse history search (like a unix terminal)
[*]TAB to cycle matches, Shift+TAB to cycle backwards
[*]Multi-line match list with current position indicator
[/list]

[size=3][b]Usage[/b][/size]
Start typing a command and a gray suggestion line appears. Press TAB to accept or cycle through matches.

[b]Examples:[/b]
[list]
[*][font=Courier New]coc [/font] + TAB - cycles through all cells in your load order
[*][font=Courier New]player.setav [/font] + TAB - cycles through actor values
[*][font=Courier New]player.getav health[/font] - shows the current value inline
[*][font=Courier New]getgs fJumpHeightMin[/font] - shows the current setting value inline
[*][font=Courier New]player.additem [/font] + TAB - cycles through inventory items
[*][font=Courier New]player.equipitem [/font] + TAB - cycles through equippable items
[*][font=Courier New]placeatme [/font] + TAB - cycles through placeable forms
[*][font=Courier New]player.addperk [/font] + TAB - cycles through perks
[*][font=Courier New]player.addfaction [/font] + TAB - cycles through factions
[*][font=Courier New]playsound [/font] + TAB - cycles through sounds
[*][font=Courier New]imod [/font] + TAB - cycles through image space modifiers
[*][font=Courier New]fw [/font] + TAB - cycles through weathers
[*][font=Courier New]cow [/font] + TAB - cycles through world spaces
[*][font=Courier New]playidle [/font] + TAB - cycles through idle animations
[*][font=Courier New]addspell [/font] + TAB - cycles through spells
[*][font=Courier New]setstage [/font] + TAB - cycles through quests, then stages
[*][font=Courier New]myquest.myvar[/font] + TAB - cycles through quest script variables
[*][font=Courier New]search "gun" [/font] + TAB - cycles through form type codes (WEAP, AMMO, etc.)
[*][font=Courier New]!myalias[/font] + TAB - cycles through defined aliases
[/list]

[b]Reverse history search:[/b]
Press Ctrl+R to search your command history. Keep pressing Ctrl+R to cycle through matches. Press TAB or Enter to accept, Ctrl+C to cancel.

[b]Command aliases:[/b]
Define shortcuts in [font=Courier New]Data\config\EnhancedConsoleAutocomplete.ini[/font]:
[code]
[Aliases]
god=tgm
ghost=tcl
heal=player.resethealth
[/code]
Type [font=Courier New]!god[/font] and press Enter to run [font=Courier New]tgm[/font]. Press TAB to cycle through matching aliases.

[size=3][b]Configuration[/b][/size]
Edit [font=Courier New]Data\config\EnhancedConsoleAutocomplete.ini[/font]:
[code]
[General]
iMatchListSize=5
[/code]
Controls the number of visible matches in the dropdown list (0-20). Set to 0 to hide the list.

[size=3][b]Notes[/b][/size]
[list]
[*]Base form loading may be slow on first use with large load orders - a warning is displayed
[*]Ctrl+TAB reloads cells and base forms from scratch (useful if you've loaded new plugins mid-session)
[/list]

[size=3][b]Installation[/b][/size]
Install with a mod manager or extract to Data folder.

[size=3][b]Compatibility[/b][/size]
[list]
[*][b]Console Paste[/b] - compatible. Disables Console Paste's autocomplete to prevent conflicts (paste functionality still works).
[*][b]Basic Console Autocomplete[/b] - [color=red]not compatible[/color]. Disable it before using this mod. Both patch the same console code.
[*][b]Improved Console (NVSE)[/b] - compatible. Both work together without issues.
[/list]
